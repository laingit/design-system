## About Checkbox-button-group

The checkbox-button-group is a set of [checkbox buttons](/components/checkbox-button) wrapped in a fieldset that is also a [form element](/components/form-element). It is described by a legend with the classes `slds-form-element__legend` and `slds-form-element__label` applied to it.

### Variations

#### Stacked

A stacked modifier that accounts for the limited width of mobile devices. It's recommended to use the stacked modifier when you have 3 or more options.

<Blockquote type="note" header="Mobile Only">
  The <code>.slds-checkbox_button-group_stacked</code> modifier only affects
  mobile devices, all other devices will show the default layout.
