## Base

The size of a button icon is based off the size of the icon that is being consumed. The default size of the consumed icon is a 16px by 16px square.

To alter the size of the icon boundary, please refer to [Icon](/modules/icon) documentation.

<CodeView toggleCode={false}>
  <React.Fragment>
    {ButtonIconExamples.default()}
    {getDisplayElementById(ButtonIconExamples.states, "default-disabled")}
  </React.Fragment>
</CodeView>

## Base with borders

The default border color of a button icon is transparent. You may apply a border color to the button icon by applying `sldswc-button-icon_bordered` to the button icon.

<CodeView toggleCode={false} style={{backgroundColor: '#f3f2f2', padding: '1rem', margin: '-1rem'}}>
  <React.Fragment>
    {getDisplayElementById(ButtonIconExamples.examples, "bordered")}
    {getDisplayElementById(ButtonIconExamples.states, "bordered-disabled")}
  </React.Fragment>
</CodeView>

## Base with fill background

The default background color of a button icon is transparent. You may apply a background fill of white by applying `sldswc-button-icon_background` to the button icon. It is **highly recommended** that a bordered is applied with the background fill.

<CodeView toggleCode={false} style={{backgroundColor: '#f3f2f2', padding: '1rem',margin: '-1rem'}}>
  <React.Fragment>
    {getDisplayElementById(ButtonIconExamples.examples, "filled")}
    {getDisplayElementById(ButtonIconExamples.states, "filled-disabled")}
  </React.Fragment>
</CodeView>
